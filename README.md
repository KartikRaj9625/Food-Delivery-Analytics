# Food Delivery Analytics Dashboard

## Project Overview
This project analyzes food delivery operations using Python and Power BI to identify factors affecting delivery performance, customer satisfaction, and workforce efficiency.

## Dataset
- Records: 41,417
- Features: 23

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Power BI

## Key Insights
- Multiple deliveries significantly increase delivery time.
- Peak-hour orders take longer than non-peak orders.
- Higher delivery ratings are associated with faster deliveries.
- Traffic congestion and adverse weather conditions cause major delays.
- Elite delivery partners were identified based on ratings, delivery time, and order volume.

## Dashboard Pages
### Executive Overview

### Operations Dashboard

### Customer Experience
